* Create multiple student objects

##  Practice Completed

* Created plain JavaScript objects
* Compared objects with classes
* Built classes using constructors
* Used `this` to initialize object properties
* Verified that objects created using `new` are independent
* Calculated discounted prices using object methods
* Explored the difference between normal functions and arrow functions with the `this` keyword

##  Key Learnings

* Classes act as blueprints for creating multiple objects.
* `new` creates a new independent object every time.
* `this` refers to the calling object in normal methods.
* Arrow functions inherit `this` from their outer scope.
* `return` sends a value back, while `console.log()` only displays it.
* OOP helps reduce code duplication and makes applications easier to maintain.
