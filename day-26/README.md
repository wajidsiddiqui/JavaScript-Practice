#  Day 26 – OOP Counter App

Today I built a simple Counter Application using Object-Oriented Programming (OOP) concepts in JavaScript.

##  Features

* Increase Counter
* Decrease Counter
* Reset Counter
* Real-time UI Updates
* Clean and Responsive Interface

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

##  Concepts Practiced

* Classes
* Constructors
* Objects
* `this` keyword
* Object Methods
* DOM Manipulation
* Event Listeners
* Updating UI Dynamically

##  What I Learned

* How to create a class and instantiate objects using `new`
* How `this` refers to the current object inside class methods
* How to separate application logic from the user interface
* How to update the DOM based on object state
* How Object-Oriented Programming makes code more organized and reusable
