# Todo List App

A simple Todo List application built using HTML, CSS and JavaScript.

## Features

- Add new tasks
- Delete tasks
- Prevent empty tasks from being added
- Dynamic DOM manipulation
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Concepts Practiced

- DOM Manipulation
- Event Listeners
- createElement()
- appendChild()
- remove()
- textContent
- Input Handling

## Project Preview

Users can:

1. Enter a task
2. Click "Add Task"
3. View tasks instantly
4. Remove tasks using the delete button
