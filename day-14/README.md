#  Book Library

A simple JavaScript project that allows users to add and remove books dynamically.

##  Features

- Add Book
- Add Author
- Delete Book
- Total Books Counter
- Input Validation
- Responsive UI

##  Technologies Used

- HTML5
- CSS3
- JavaScript (DOM)

##  Concepts Practiced

- querySelector()
- addEventListener()
- createElement()
- appendChild()
- remove()
- classList
- textContent
- value
- trim()
