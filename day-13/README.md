#  Digital Stopwatch

A simple Digital Stopwatch built using HTML, CSS, and JavaScript.

##  Features

- Start Timer
- Stop Timer
- Reset Timer
- Milliseconds Support
- Responsive UI
- Clean Design

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Concepts Practiced

- DOM Manipulation
- querySelector()
- addEventListener()
- setInterval()
- clearInterval()
- Variables
- Functions
- Conditions
- String.padStart()
