# Download Progress Bar

A simple JavaScript project that simulates a file download process using a progress bar.

## Features

* Download button to start the process
* Animated progress bar
* Percentage updates in real time
* Random download speed simulation
* Button disabled while downloading

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## Concepts Practiced

* DOM Selection
* Event Listeners
* setInterval()
* clearInterval()
* Math.random()
* Dynamic Style Updates
* Progress Bar Animation

## How It Works

1. Click the **Download** button.
2. A random download speed is generated.
3. The progress bar starts filling.
4. The percentage increases from 0% to 100%.
5. Once completed, the download process stops.

## Learning Outcome

This project helped me practice JavaScript DOM manipulation, timers, event handling, and dynamic UI updates.
