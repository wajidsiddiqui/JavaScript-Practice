# Character Counter

A simple Character Counter built using HTML, CSS and JavaScript.

## Features

- Live Character Count
- Live Word Count
- 200 Character Limit
- Border changes to red when the limit is reached

## Technologies

- HTML
- CSS
- JavaScript

## Concepts Practiced

- DOM Manipulation
- querySelector()
- addEventListener()
- input Event
- value
- textContent
- String Methods
- Conditional Statements
