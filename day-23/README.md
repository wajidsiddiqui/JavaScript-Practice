# Day 23 - Matrix Spotlight Reveal

A futuristic Matrix-inspired animation built using HTML, CSS, and JavaScript.

##  Features

-  Mouse-following spotlight effect
-  Neon Matrix-inspired UI
-  Character scramble reveal animation
-  Replay animation button
-  CSS Glow Animations
-  Linear & Radial Gradients
-  Responsive-friendly layout

- CSS3
- JavaScript (ES6)
