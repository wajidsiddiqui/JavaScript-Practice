# Day-19 - Project 2: Add Friend Feature

## Description

A simple DOM manipulation project where a user can add or remove a friend by clicking a button.

When the button is clicked:

- Stranger ➜ Friends
- Add Friend ➜ Remove
- Status color changes from Crimson to Green

Clicking again toggles everything back to the original state.

---

##  Features

- Toggle Friend Status
- Toggle Button Text
- Change Status Color
- DOM Manipulation
- Event Handling
